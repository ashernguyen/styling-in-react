# HOW TO STYLE IN REACT?

---

## Stylesheet (Css, Sass & Less)

```scss
.button {
  display: 'inline-flex';
  background-color: '#342f4d';
  color: '#ffffff';

  &__content {
    font-size: '0.875rem';
  }
}
```
---

## Stylesheet (Css, Sass & Less)

```jsx
import './button.scss'

const StyledButton = ({ handleClick, children }) => (
  <button className="button" onClick={handleClick}>
    <p className="button__content">{children}</p>
  </button>
)
```

---

## Inline Styling

```jsx
const buttonStyle = {
  display: 'inline-flex',
  backgroundColor: '#342f4d',
  color: '#ffffff'
}

const ContentStyle = {
  fontSize: '0.875rem'
}

const StyledButton = ({ handleClick, children }) => (
  <button style={buttonStyle} onClick={handleClick}>
    <p style={ContentStyle}>{children}</p>
  </button>
)
```

---

## CSS Modules

```jsx
:local(button) {
  display: 'inline-flex';
  background-color: '#342f4d';
  color: '#ffffff';
}

:local(content) {
  font-size: '0.875rem';
}
```

---

## CSS Modules

```jsx
import styles from './Button.css';

const StyledButton = ({ handleClick, children }) => (
  <button className={styles.button} onClick={handleClick}>
    <p className={styles.content}>{children}</p>
  </button>
)
```

## CSS-in-JS (styled-component | emotion  | jss)

```jsx
import styled, { ThemeProvider } from 'styled-components';

const theme = {
  colors: {
    blueSky: '#93afff',
    lightOrange: '#ffc097'
  }
}

const Button = styled.button`
  display: 'inline-flex';
  background-color: `${props => props.theme.colors.blueSky}`;
  color: `${props => props.theme.colors.lightOrange}`;
  border-color: `${props => props.customBorderColor}`
`

const Content = styled.p`
  font-size: '0.875rem';
`

const StyledButton = ({ handleClick, children }) => (
  <ThemeProvider theme={theme}>
    <Button onClick={handleClick} customBorderColor="#fdsafd">
      <Content>{children}</Content>
    </Button>
  </ThemeProvider>
)
```

## Styled-system

```jsx
import { space, width, fontSize, color } from 'styled-system';
import styled, { ThemeProvider } from 'styled-components';
import theme from './theme';

const Box = styled.div(
  {
    boxSizing: 'border-box'
  },
  space,
  width,
  fontSize,
  color
)

render(
  <ThemeProvider theme={theme}>
    <Box p={[3, 4, 5]} bg="blueSky" color="lightOrange">
      This is a Box
    </Box>
  </ThemeProvider>,
);
```

## Styled-system

**- Design Constraints Theme -**

```jsx
// font-size: 24px (theme.fontSizes[4])
<Box fontSize={4} />

// margin: 16px (theme.space[3])
<Box m={2} />

// color: #93afff (theme.colors.blueSky)
<Box color="blueSky" />

// line-height: 1.5 (theme.lineHeights.copy)
<Box lineHeight="copy" />

// renders CSS `50%` width since it's not defined in theme
<Box width={1/2} />
```
